Skew

The skew () function allows distortion by stretching each point of the element by a certain angle in a direction of the plane. For this, we increase the ordinate of an element of a value proportional to the given angle and the distance of the origin. The further the point is from the origin, the greater the offset obtained.

Rotate

The rotate () function defines a transformation that moves an element around a fixed point (defined by the transform-origin property) without deforming it (that is, applying plane rotation). It's a rotation around this point. By default, this origin corresponds to the center of the element.

Translate

The translate() CSS function repositions an element in the horizontal and/or vertical directions. Its result is a transform-function data type. This transformation is characterized by a two-dimensional vector. Its coordinates define how much the element moves in each direction.
