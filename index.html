Responsive image pages

Adaptive Images is the practice of adapting the size, resolution and quality of the images used in the editorial content of a web page to the user. Adaptive images are necessary for Responsive sites that will adapt their display to the consultation device : on a Responsive site the same page is rendered differently depending on the size of the screen. Adapting images is all about improving the user experience by delivering the image that's best for you: you do not need to send an image that is too big and too heavy if the user views the page on a smartphone with a small screen and bad internet connection!

Flexbox Demo

FlexBox is a layout mode that provides a layout of the elements of a page so that these elements have predictable behavior when they have to accommodate different sizes of screens / devices. In many cases, the Flexbox box model offers an enhancement of the block model in which floats are not used, nor does the blending of flexible container margins with its elements.

Transforms
